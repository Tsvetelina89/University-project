# University Project: Employee Management System
Конзолно приложение на C++ за управление на данни за служители и изчисляване на статистики за заплати.
## Функционалности
*   **Добавяне на служители:** Въвеждане на професия, заплата и пол с вградена проверка за лимит и коректност на данните.
*   **Статистика за заплати:** Изчисляване на средна заплата за жени по професии.
*   **Най-високоплатени служители:** Намиране на жената с най-висока заплата в компанията.
*   **Преглед на данни:** Пълен списък на всички регистрирани служители.
## Технологии
*   **Език:** C++
*   **Структура:** Използване на "struct", "enum", хедър файл (.h) и разделяне на логиката в отделни модули.
